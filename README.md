# CivicConnect (JanSahyog)

A smart city civic issue reporting platform built with React + Vite.

## Features
- **Report Issues**: Citizens can report problems like potholes, garbage, etc.
- **Explore**: View reported issues in your area.
- **Responsive Design**: Premium green-themed UI using vanilla CSS.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Ansh-Yadav07/sih.git
    cd sih
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run development server:
    ```bash
    npm run dev
    ```

## Technology Stack
-   React
-   Vite
-   Vanilla CSS (Custom Design System)
-   React Router DOM
-   Lucide React (Icons)
